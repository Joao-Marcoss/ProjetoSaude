O que é IMC?

O Índice de Massa Corporal (IMC) é uma medida que relaciona o peso e a altura de uma pessoa para avaliar se seu peso está dentro de faixas consideradas saudáveis. Calculado dividindo o peso (em quilogramas) pela altura (em metros) ao quadrado, fornece uma indicação geral do estado nutricional, com faixas que sugerem subnutrição, peso saudável, sobrepeso ou obesidade.
